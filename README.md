# Tandem

### Prerequisites (globally installed)

 - node + npm
 - mongod (not atm but will)
 - yarn
 
After installing these, clone the repository. Within its contents run:
```
yarn install
```

### Website/Webapp 

To run:
```
yarn web-app
```

Uses:

- ES6 syntax
- React + Reflux

### Backend

To run:
```
yarn backend
```

Uses:

- express
- mongoose/mongo
